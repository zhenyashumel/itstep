Создать класс Logger, который в файл записывает (метод WriteLog) информацию о действиях пользователя, возникших ошибках и исключениях. 
Данный класс должен производить запись в файл из различных потоков. 
Для корректной работы класса Logger необходимо реализовать синхронизацию потоков.
Проверить работоспособность данного класса. 

Дополнительно:
1. Реализовать синхронизацию с применением различных средств (lock, Monitor,  Mutex, Semaphore, Events)
2. Оценить быстроденйствие каждого из средств. 

Формат записи в лог-файл:
07.03.2018 |  12:11:23.006 | Ok        | Подключение к серверу
07.03.2018 |  12:11:24.125 | Error     | Ошибка  чтения данных
07.03.2018 |  12:11:25.423 | Exception | Файл заблокирован другим процессом
07.03.2018 |  12:11:27.256 | Ok        | Откоючение от сервера
