
Целью проекта является создание десктопного клиента для https://www.themoviedb.org/
Данный портал предоставляет API работы со своей базой. 
Существует несколько .Net-библиотек для работы с данным API. 
https://www.themoviedb.org/documentation/api/wrappers-libraries
Рекомендуется использовать The Movie Database .NET Client (исходники примеры доступны на github). Для использования API необходимо зарегистрироваться на https://www.themoviedb.org/ и запросить API Key.

Проект необходимо разрабатывать на WPF с использованием паттерна MVVM. 
Необходимо использовать какой-либо IoC-контейнер 
(например, Autofac, Unity, Ninject и т.п)
Приложение должно содержать следующие экраны:
- Списки самых популярных, самых новых и идущих в данный момент фильмов и сериалов
- Поиск фильмов (по названию, актеру, жанру, режиссеру и т.п.) (экран поиска и результаты)
- Поиск сериалов
- Поиск актеров
- Отображение детальной информации о фильме (постер, сюжет, список актеров и т.п.
- Отображение информации об актере, включая фильмографию
- Отображение трейлера фильма

Пользовательский интерфейс спроектировать по собственному усмотрению. 
- Приложение должно корректно обрабатывать ошибки (например, пропадание сети). 
- Приложение не должно падать. 
- Приложение должно валидировать пользовательский ввод. 
- Приложение не должно подвисать при запросах данных у сервера и прочих длительных операциях.
